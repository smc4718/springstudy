10월 18일 수업정리 ( build ) 
1. 빌드와 배포
  1) 빌드 : 프로젝트를 배포하기 위해서 하나의 파일로 만드는 과정 (war 또는 jar)
  2) 배포 : deploy, WAS(Tomcat)에 빌드된 파일을 배포하는 것

2. 빌드
  1) Run as - Maven Build
  2) 빌드 파일명 (pom.xml에서 확인)
      (1) artifactId-version.packaging
    (2) 사용자 지정
         <build>
         </finalName>${artifactId}</finalName>
    (3) 빌드 파일 생성 경로
        프로젝트\target
